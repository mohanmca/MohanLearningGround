  * c is a fixed point of the function f(x) if f(c) = c.
  * if c is a fixed point, wherever you pass f(c), you could just pass c. (Stopy applying function)
  * Below someCurve has fixed point of 2
  * ```const someCurve = (n) =>  n ^2 - (3 * n) + 4```
